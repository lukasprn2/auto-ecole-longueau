Base de données Firebase
Toutes les modifications (horaires, contacts, tarifs, FAQ, identifiants) sont synchronisées avec Firebase Realtime Database quand la connexion est configurée.
Structure des données Firebase :
autoecolelongueau/
├─ horaires/ ← jours + heures ouverture
├─ contacts/ ← tel, email, adresse, fb
├─ tarifs/ ← toutes les prestations
├─ faq/ ← questions / réponses
└─ auth/ ← username + hash mot de passe

Pour connecter Firebase, remplacez les valeurs VOTRE_* dans la configuration en haut du fichier HTML.
